# 📊 Food Delivery Performance Analysis

## 📌 Project Overview
This project analyzes online food delivery data to understand order trends, delivery efficiency, and restaurant performance.

## 🎯 Objectives
- Analyze peak order times
- Evaluate delivery performance
- Identify high-performing restaurants
- Improve operational efficiency using insights

## 🛠️ Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)
- Google Colab
- Power BI

## 🔄 Project Workflow
1. Data Collection
2. Data Cleaning (Python - Colab)
3. Feature Engineering (Hour, Time Slot, Delivery Status)
4. Data Visualization (Power BI Dashboard)
5. Insights Generation

## 📊 Dashboard Features
- KPI Metrics (Total Orders, Avg Delivery Time, Fast Deliveries %)
- Time-based analysis (Hourly trends)
- Restaurant performance analysis
- Delivery efficiency insights

## 🔍 Key Insights
- Peak orders occur during evening hours  
- Most deliveries are completed within 30–45 minutes  
- Delivery time increases with distance  
- Motorbikes provide faster delivery performance  
- High-demand restaurants show slight delivery delays  

## 📁 Files Included
- food_delivery_cleaned.csv
- analysis.ipynb (Google Colab)
- dashboard.pbix

## 🚀 Conclusion
This project demonstrates how data analytics can improve decision-making in food delivery systems by identifying inefficiencies and optimizing performance.
